# Paraglider Stamina Replacer

An override for Tictim's Paragliders mod that replaces the default stamina circle with a more vanilla-friendly HUD bar.

## Features

- Removes the circular stamina wheel from the original mod
- Adds a configurable horizontal stamina bar, similar to the vanilla health bar
- Supports in-game drag-and-drop positioning for personal preference
- Respects screen resizing and UI scale

## Requirements

- Minecraft 1.20.1
- Forge 47.4.13
- [Tictim's Paragliders mod](https://www.curseforge.com/minecraft/mc-mods/paragliders)

## Installation

1. Make sure you have Tictim's Paragliders installed.
2. Place the `Paraglider Stamina Replacer` mod JAR into your `mods` folder.
3. Launch Minecraft with Forge.
